AESEncrypt.cs
{
	文字列を共通鍵暗号方式(AES)によって暗号化・復号化するスクリプトです。

	Encrypt:
		平文を暗号化します。

	Decrypt:
		暗号文を復号化します。
}

CSVLoader.cs
{
	CSVファイルを外部から読み込む際に用いるスクリプトです。

	Load:
		ファイルパスを指定してCSVの読み込みを行いデータを格納します。

	Unload:
		CSVデータを格納する変数の内容を初期化します。

	GetCols:
		CSVの列数を返します。

	GetRows:
		CSVの行数を返します。

	GetDataCount:
		CSVのデータ数を返します。

	GetString:
		指定行列のデータを文字列型で返します。

	GetInteger:
		指定行列のデータを整数型で返します。

	GetFloat:
		指定行列のデータを実数型で返します。
}

Range.cs
{
	最小から最大の実数値の範囲を格納する構造体です。
	インスペクタでの編集がしやすいよう専用のエディタ拡張を組んであります。

	Range:
		コンストラクタです。第一引数でmin(最小値), 第二引数でmax(最大値)を設定します。
		第一引数と第二引数を誤って反対に書いた場合の対応策として、設定時に両引数でより小さい値を動的に最小値に格納します。

	IsIn:
		引数値が範囲内かどうかを返します。

	GetIn:
		引数値を範囲内に収束した値を返します。
		たとえば Range(0, 255) において、引数96の場合は96が、引数280の場合は255が返ります。

	GetOut:
		引数値を範囲内に発散した値を返します。
		たとえば Range(0, 255) において、引数96の場合は0が、引数130の場合は255が、引数280の場合は280が返ります。

	GetCompress:
		古い範囲から新しい範囲に拡縮した際の値を取得します。
		たとえば Range(0, 100) において 引数 40, Range(100, 600) の場合は300が返ります。

	GetAround:
		引数値を範囲内に収束した値を返します。ただし、超えている分はループ(min⇔max)として扱います。
		たとえば Range(10, 100) において、引数162の場合は52が、引数5の場合は95が返ります。

	DebugLog:
		最小値と最大値をデバッグログに残します。
}

Timer.cs
{
	時間を計測する構造体です。
	3秒ごとに１フレームだけ処理・８秒間の間処理し続ける、といった際に活用を推奨いたします。

	Setup:
		引数の秒数を計るタイマーとして設定されます。
		それまでに設定していた時間はリセットされます。

	Reset:
		タイマーの終了時間を変更せずに、経過秒数だけをリセットします。

	Update:
		タイマーの更新を行います。毎フレーム実行してください。
		引数をfalseにすることでTime.timeScaleの影響を受けずに更新します。(デフォルトではtrue)

	Fin:
		実行することでタイマーの経過秒数を強制的に終了時間にします。

	IsFinished:
		タイマーの経過時間が終了時間に達しているかを返します。

	DebugLog:
		経過時間と終了時間をデバッグログに残します。
}
