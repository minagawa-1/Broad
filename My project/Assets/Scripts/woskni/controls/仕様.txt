KeyBoard.cs
{
	キーボードの複数押し込みに関するスクリプトです。
	コマンド入力を用いる場合は、コンポーネントとしてアタッチする必要があります。

	GetOrKey:
		複数キーのOR検知です。
		たとえば引数に KeyCode.S, KeyCode.Alpha1, KeyCode.Alpha2 を入れた場合、S,1,2のいずれかが押されている間は常にtrueを返し続けます。

	GetOrKeyDown:
		複数キーのOR検知です。
		たとえば引数に KeyCode.S, KeyCode.Alpha1, KeyCode.Alpha2 を入れた場合、S,1,2のいずれかが押されたフレームにtrueを返します。

	GetOrKeyUp:
		複数キーのOR検知です。
		たとえば引数に KeyCode.S, KeyCode.Alpha1, KeyCode.Alpha2 を入れた場合、S,1,2のいずれかの押し込みが解除されたフレームにtrueを返します。
	
	GetAndKey:
		複数キーのAND検知です。
		たとえば引数に KeyCode.S, KeyCode.Alpha1, KeyCode.Alpha2 を入れた場合、S,1,2の全てが押されている間は常にtrueを返し続けます。

	GetAndKeyDown:
		複数キーのAND検知です。
		たとえば引数に KeyCode.S, KeyCode.Alpha1, KeyCode.Alpha2 を入れた場合、1,2が押されている間にSを押されたフレームにtrueを返します。

	GetAndKeyUp:
		複数キーのAND検知です。
		たとえば引数に KeyCode.S, KeyCode.Alpha1, KeyCode.Alpha2 を入れた場合、1,2が押されている間にSの押し込みが解除されたフレームにtrueを返します。
	
	IsCommandedKeyDown:
		コマンド入力検知です。
		たとえば引数に KeyCode.S, KeyCode.Alpha1, KeyCode.Alpha2 を入れた場合、S → 1 → 2 の順で押されて、引数最後尾の2が押されたフレームにtrueを返します。
		コマンド入力中に別のキーが押された場合、コマンド入力はリセットされます。

	IsCommandedKeyUp:
		コマンド入力検知です。
		たとえば引数に KeyCode.S, KeyCode.Alpha1, KeyCode.Alpha2 を入れた場合、S → 1 → 2 の順で押されて、引数最後尾の2の押し込みが解除されたフレームにtrueを返します。
		コマンド入力中に別のキーが押された場合、コマンド入力はリセットされます。

	LogCommandList:
		入力されたキーを引数の数だけ新しい順に表示します。
}

KeyRepeat.cs
{
	キーを長押ししているとき、一定間隔でtrueを返す関数を格納したスクリプトです。
	たとえば、アイテム一覧UIの十字キーカーソル移動で活用可能です。

	KeyRepeat:
		コンストラクタです。引数にKeyCodeを設定します(以下コンストキー)。
	
	Repeat:
		たとえば引数に 1f, 0.2f, true を入れた場合、コンストキーが押されたフレームとそのフレームの1秒後から0.2秒ごとにtrueを返し続けます。
		コンストキーの押し込みが解除されたとき、カウントタイマーは初期化されます。
		第三引数のBoolean型引数は、Time.timeScaleの影響を受けるか否かです。
}
